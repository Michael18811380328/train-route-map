# 火车运行模拟器产品文档

## 1. 产品概述

火车运行模拟器是一个可视化展示火车运行系统的工具，允许用户查看、管理和模拟火车线路的运行情况。

产品分为两个阶段实现：

- **第一阶段**：实现静态模拟，展示预定义的站点、路线和车次信息
- **第二阶段**：增加用户管理功能，支持对站点、路线和车次的增删改查操作

## 2. 功能需求

### 第一阶段：静态模拟

#### 2.1 站点信息管理
- 展示站点列表（名称、ID、位置坐标）
- 在地图上可视化所有站点
- 支持点击站点查看详情信息

#### 2.2 路线信息管理
- 展示所有预定义路线（起点、终点、途径站点）
- 在地图上可视化路线路径
- 显示路线详细信息（长度、预计运行时间）

#### 2.3 车次信息管理
- 展示所有车次信息（车次号、车型、运行路线）
- 模拟车次在路线上的运行情况

#### 2.4 模拟运行功能
- 时间轴控制（播放/暂停/重置）
- 模拟速度调节（1x, 2x, 5x）
- 实时显示当前模拟时间
- 在地图上实时展示车次位置

### 第二阶段：动态管理

#### 2.5 站点管理
- 添加新站点（名称、坐标）
- 编辑现有站点信息
- 删除站点（需验证是否被路线使用）

#### 2.6 路线管理
- 创建新路线（选择起点、终点和途径站点）
- 编辑现有路线信息
- 删除路线（需验证是否被车次使用）

#### 2.7 车次管理
- 添加新车次（选择路线、设置发车时间）
- 编辑车次信息
- 删除车次

### 数据库设计

#### 站点模型 (Station)
```typescript
class Station {
  constructor(obj) {
    this.id = obj.id;
    this.x = obj.x;
    this.y = obj.y;
    this.name = obj.name;
    this.size = obj.size; // 表示车站等级
  }
}
```

#### 路线模型 (Route)
```typescript
class Route {
  constructor(obj) {
    this.id = obj.id;
    this.name = obj.name;
    this.color = obj.color || 'black'; // 线路颜色，表示普铁动车高铁线路
    this.speed = obj.speed || 100; // 线路最大运行速度
    this.line_number = obj.line_number; // 线路个数，表示单线复线等
    this.points = obj.points; // 站点ID数组
  }
}
```

#### 车次模型 (Train)
```typescript
class Train {
  constructor(obj) {
    this.id = obj.id;
    this.train_number = obj.train_number; // 车次号 G101
    this.start = obj.start; // 开始时间
    this.end = obj.end; // 结束时间
    this.color = obj.color; // 车次颜色（表示不同等级）
    this.times = obj.times; // 车次途径的时间
    this.stations = obj.stations; // 车次在某个时间途径的站点
  }
}
```

### 4.3 API设计

#### 站点API
- `GET /api/stations` - 获取所有站点
- `GET /api/stations/:id` - 获取单个站点
- `POST /api/stations` - 创建新站点
- `PUT /api/stations/:id` - 更新站点
- `DELETE /api/stations/:id` - 删除站点

#### 路线API
- `GET /api/routes` - 获取所有路线
- `GET /api/routes/:id` - 获取单个路线
- `POST /api/routes` - 创建新路线
- `PUT /api/routes/:id` - 更新路线
- `DELETE /api/routes/:id` - 删除路线

#### 车次API
- `GET /api/trains` - 获取所有车次
- `GET /api/trains/:id` - 获取单个车次
- `POST /api/trains` - 创建新车次
- `PUT /api/trains/:id` - 更新车次
- `DELETE /api/trains/:id` - 删除车次

#### 模拟API
- `GET /api/simulation` - 获取当前模拟状态
- `POST /api/simulation/start` - 开始模拟
- `POST /api/simulation/pause` - 暂停模拟
- `POST /api/simulation/reset` - 重置模拟
- `PUT /api/simulation/speed` - 设置模拟速度
